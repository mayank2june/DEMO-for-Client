#  React + Storybook + Redux Three.js Component Library

This repository demonstrates a modern front-end development workflow using **React**, **Storybook**, **Redux Toolkit**, and **Three.js / React-Three-Fiber**.  
It showcases reusable UI components, component documentation, state management, and an interactive 3D scene example.

---

## Project Highlights

## Component-Driven Development with Storybook
- UI components are built and tested in isolation
- Support for CSF stories, Controls, Args & Docs addon
- Ready for design handoff and scalable UI architecture

### Redux Toolkit State Management
- Global store setup example
- Slice-based modular structure

###  3D Graphics Integration using Three.js
- React-Three-Fiber renderer
- Orbit controls & animated 3D objects
- Smooth UI & 3D component interaction
